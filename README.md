# yolov5-v1-chinese-comment

yolov5 v1版本中文注释

yolov5的整体训练流程如下图所示，本仓库主要对模型构建以及损失函数计算部分进行了详细地注释，希望对新手有帮助。
所有的注释文档都放在代码解读文件夹，最核心的文件是“yolo.py解析”和“loss计算”这两个文件夹，分别对应模型构建和loss计算部分。


建议：这些注释是在yolov5中提供的coco128数据上进行的调试，建议一边调试一边对照着看文档。
